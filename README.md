# BigEye
<p>这是一个跟视觉有关的组件库</p>

## 目的
一个是体验方面的习作，一个是自制一个源生的视觉效果池，主要面向移动平台，但是会主动支持桌面平台的现代浏览器，也就是说ie9以下基本是完全不支持的。
