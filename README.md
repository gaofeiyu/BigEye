# BigEye
<p>这是一个跟视觉有关的组件库</p>

## 目的
一个是体验方面的习作，一个是自制一个源生的视觉效果池，主要面向移动平台，但是会主动支持桌面平台的现代浏览器，也就是说ie9以下基本是完全不支持的。

## 开发进度
<p>这里是任务池，想到就添加，标注已完成的是可用的，点击对应组件可以看到详细进度。</p>
<ul>
    <li>show360:物体360°拖动展示[已完成基本功能]</li>
    <li>tween: 缓动组件[已完成基本功能]</li>
</ul>
